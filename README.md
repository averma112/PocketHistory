Pure JavaScript Next.js (App Router) starter for PocketHistory.
- No TypeScript. All files are `.js`/`.jsx`.
- Tailwind, SWR, Recharts.
- Local storage layer for instant MVP.
- Modal uses React Portal with high z-index to avoid chart overlap.
- MongoDB API stubs ready (`MONGODB_URI`).

## Run
npm install
npm run dev

## Enable Mongo
Copy `.env.example` to `.env.local` and set `MONGODB_URI`.
